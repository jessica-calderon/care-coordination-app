# Care Coordination App

**Work in Progress**

A calm, simple web application for family caregivers to coordinate daily care responsibilities. This app helps families keep track of care notes, symptoms, tasks, and handoffs without the stress of complex tools.

## Features

- **Daily care notes** - Keep track of care, symptoms, and important observations throughout the day
- **Today's view** - See what matters today at a glance
- **Caregiver handoffs** - Smoothly transition care responsibilities between family members
- **Task tracking** - Keep track of what needs to be done next
- **Shared access** - All caregivers can view and update the same care notebook

Built with React, TypeScript, and Tailwind CSS.
